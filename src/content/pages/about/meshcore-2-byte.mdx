<section id="meshtastic-vs-meshcore">
  <hgroup>
    <h1>MeshCore 2-byte vs 1-byte paths</h1>
    <p>Every node(radio) on the network gets a nickname. In 1-byte mode that's something like `a1`. In 2-byte mode that's `a1b2`. 1-byte conflicts at 256 unique nicknames while 2-byte has 65,536.</p>
  </hgroup>
  <h2>Explaining key-prefixes</h2>
  <p>Every node in the mesh has a Public Key. The first characters of that key are treated as the address so other nodes know how to route messages. With 1-byte paths(the first 2 characters of the public key), there are only 256 possible nicknames to go around, so as the network grows, more and more nodes end up sharing the same one. When that happens, messages can get handed off to the wrong node and routing breaks down.</p>
  <p>2-byte paths use the first 4 characters of the public key, allowing 65,536 possibilities instead of 256. More room, fewer mix-ups, much healthier mesh as it grows.</p>
  <p><strong>The catch(es)</strong>: every repeater between you and the person you're messaging has to understand 2-byte nicknames, or it'll drop the message. That means firmware upgrades. It also reduces the max number of hops from 64 to 32.</p>

  <h2>Requirements</h2>
  <ul>
    <li><strong>Repeaters:</strong> firmware <strong>1.14 or later</strong>. Older repeaters can't forward 2-byte paths.</li>
    <li><strong>Companions:</strong> firmware <strong>1.14 or later</strong>, plus the latest phone app (the toggle lives under Experimental Settings).</li>
  </ul>
  <p><em>If any repeater in the path is on older firmware, 2-byte traffic won't make it through</em>. Plan upgrades accordingly.</p>
  <h2>Turning it on</h2>
  <h3>Repeaters</h3>
  <ol>
    <li>Upgrade to at least 1.14</li>
    <li>Connect to the repeater's console and run: <pre><code>set path.hash.mode 1</code></pre></li>
  </ol>
  <h3>Companions</h3>
  <p>Companions need an extra step to actually <em>send</em> messages using 2-byte paths:</p>
  <ol>
    <li>Confirm the companion is on firmware 1.14 or later.</li>
    <li>Open the phone app.</li>
    <li>Go to <strong>Experimental Settings</strong>.</li>
    <li>Set <strong>Default Path-Hash Size</strong> to 2-byte (max 32 hops).</li>
  </ol>
  <p>Once that's on, messages from the companion will use 2-byte paths when available.</p>
</section>
