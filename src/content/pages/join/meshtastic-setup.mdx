<hgroup>
  # Meshtastic Setup
</hgroup>

<section>
  ## 1) Hardware/Radio

  You'll need a radio to get started. Prices range from $13 - hundreds. Many sit in the $30-80 range.

  Find our [suggested hardware on the devices page](/devices/#options).

  ## 2) Software Setup

  1. Flash the Meshtastic firmware to your radio using <a href="https://flasher.meshtastic.org/" rel="nofollow noopener">flasher.meshtastic.org</a>.
  2. Download the Meshtastic app to your phone.
  3. Connect to your radio via bluetooth.
  4. Select the **United States** region.
  5. Select the **Long Range - Fast** preset.
  6. **Turn on**: Ignore MQTT, OK to MQTT
  7. **Do not adjust** frequency slot, frequency override.
  8. Set your Long Name and Short Name. These will be visible to others on the network.

  ## 3) Channels and Use

  1. Austin Mesh uses the default **Primary Channel** that is automatically configured.
  2. Send your first message in **Public**. Does it say "Acknowledged"? If so, that means you're connected!
     - If it says "Max Retransmission Reached" try moving your radio. The higher/fewer obstructions the better.
  3. **Send away**! It's very unlikely you'll bother anyone by being chatty or saying hello a bunch.
  4. Consider [joining the Discord server](https://discord.gg/6a5Sv2s9bG) for more advice.
</section>
